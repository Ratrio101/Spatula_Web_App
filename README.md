## 📖 О проекте
> ⚠️ ** Примечание:** Этот проект был создан в 2024 году и отражает практику программирования и знания, которыми я обладал на тот момент.

Поварёшка - это полнофункциональное веб-приложение, которое я разработал в рамках своей университетской курсовой работы. Хотя проект больше не поддерживается в активном режиме и демонстрационная версия недоступна, это важная веха на моем пути развития.

В приложении реализован нетрадиционный подход к визуализации данных / взаимодействию с пользователем / обновлениям в режиме реального времени, который был довольно инновационным для своего времени.

## 🚀 Начало работы

### Предварительные требования

- VS Code (или любая другая среда разработки, поддерживающая JS)
- Node.js (версия 18 или выше)
- MongoDB (локальная или Atlas)
- npm

### Установка

1. Клонируем репозиторий:
```bash
git clone https://github.com/Ratrio101/Spatula_Web_App.git
cd Spatula_Web_App
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите приложение:
```bash
cd server
npm start

# запуск фронтенда (в другом терминале)
cd ../client
npm start
```

Приложение будет доступно по ссылке http://localhost:3000
